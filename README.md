# SmartCents

SmartCents is a financial literacy simulation game built with **Python** and **Pygame**.  
The game helps players practice budgeting, saving, and long-term planning through interactive, year-by-year decision making.

---

## How It Works
- Each turn represents **one year**
- Manage:
  - Checking account
  - Savings account (earns interest)
  - Income and upgrades
- Random financial events add real-world uncertainty
- **Win:** Reach $50,000 net worth  
- **Lose:** Stay in negative balance too long

---

## Why SmartCents
SmartCents was created to help young adults understand:
- Budgeting and saving habits
- The impact of unexpected expenses
- Long-term effects of financial decisions

Learning happens through gameplay, not lectures.

---

## Tech Stack
- **Python 3**
- **Pygame 2.6.0**
- Desktop application
